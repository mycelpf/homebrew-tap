# mycelpf/homebrew-tap

Homebrew tap for [Cogniworks Private Limited](https://cogniworks.io)'s
`mycel_*` family of CLIs.

## Install

```bash
brew install --cask mycelpf/tap/mycel
```

## What's here

Cask definitions under `Casks/` are generated and pushed automatically by
[GoReleaser](https://goreleaser.com) on every tagged release of the
corresponding source repo — see [mycelpf/mycel_cli](https://github.com/mycelpf/mycel_cli).
Don't hand-edit files under `Casks/`; they're overwritten on the next release.
